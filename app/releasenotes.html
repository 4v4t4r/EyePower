New Features:

  • Integrated Alchemy API for Image Tagging.

  • Uploading and Processing is now 2X Faster.

Added In-App Accesibilty Shortcuts:

  1. If you want to add an Image URL press Control + 1. Then type or paste the URL and Press Enter.
  2. If you want to choose an Image from your PC press Control + 2 and press Enter to show The Open File Dialog and select the Image.
  3. If you want to show this text again press Control + H in the application.
